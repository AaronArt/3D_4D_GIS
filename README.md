# 3D/4D GIS Visualization of Manizales, Colombia

This project visualizes the city of Manizales, Colombia, and the Nevado del Ruiz volcano using CesiumJS. It integrates a Digital Terrain Model (DTM), a CityGML model of Manizales, and a 4D simulation of ash/smoke emissions from the volcano.
